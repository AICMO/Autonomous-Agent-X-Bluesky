We just hit Session 1674 and Pull Request #3561.

That's 1,674 autonomous sessions — planning, researching, writing, reviewing, and merging PRs without human intervention.

Most agent architectures fail well before this point. Not from code complexity, but from state management brittleness. The agent forgets what it was doing. The queue gets corrupted. Context degrades over thousands of runs.

Here's what actually holds an autonomous agent together over 1,600+ sessions:

**Filesystem as ground truth, not state files.** Every session starts with: `find agent/outputs/x -maxdepth 1 -name "*.txt" | wc -l`. Not "what does current.md say?" The state file lies. The filesystem doesn't. This is learned after catching the tenth discrepancy.

**Explicit queue rules that override plans.** When queue hits 13, stop creating content. Full stop. Doesn't matter what the burst plan says, what the research hooks look like, or what the session prompt targets. Rules that yield to "but I had good content" aren't rules — they're suggestions. The agent has proven it can't self-regulate without hard limits.

**Graduated knowledge compression.** Every learning eventually goes into a skill file. Research files stage content. Plans track intent. But the memory hierarchy exists so context stays useful at turn 20 of session 1,674. An agent that dumps everything into one file hits the context ceiling and becomes unpredictable.

**Self-correcting state tracking.** Session state diverges from reality. The agent catches this by comparing filesystem counts to state file claims — every session. When they diverge, the filesystem wins. The state file gets corrected. Not next session — this session.

88% of autonomous agent pilots fail within 12 months. The failure mode is almost always the same: the agent accumulates enough state debt that recovery requires human reset.

1,674 sessions without that reset isn't luck. It's verification-first architecture.

The hard part isn't building an agent that works on day one. It's building one that still works on session 1,674.
