GPT-4-quality output cost $30 per million tokens in 2023.

Today: under $0.10 per million tokens in some configurations. That's a 300x reduction in roughly 30 months. 10x per year, sustained.

The natural assumption: AI bills should be plummeting.

They're not. Enterprise AI spend is rising. Some reports put it at 4x over the same period.

Here's the mechanism: agentic workflows changed the denominator.

In 2023, you ran 1 LLM call per user query. Maybe 2. A simple chatbot: user sends message → model responds → done. Low call counts, predictable cost.

Now: user sends message → planning agent decomposes it into subtasks → tool-use agent executes subtask 1 (read data) → tool-use agent executes subtask 2 (search) → synthesis agent combines results → quality agent checks output → response agent formats answer → monitoring agent logs the interaction. That's 6-8 LLM calls for a single user action. Some enterprise agentic pipelines run 15-20 calls per interaction.

RAG architectures compound this. Context windows inflated 3-5x with retrieved chunks. Token counts per call go up 300-500%. Multiply that by 10-20 calls per interaction.

Add: always-on monitoring agents. Background classification agents. Parallel evaluation runs. Agents checking other agents.

Per-token price goes down 10x. Call count goes up 20x. Context per call goes up 4x. Net: enterprise bill goes up 8x.

This is the Jevons Paradox applied to LLM inference. Efficiency gains trigger usage expansion. The cheaper it gets to compute, the more compute you use.

The companies winning at AI economics right now are the ones tracking cost-per-business-outcome, not cost-per-token. Cost-per-token is a distraction when your denominator is exploding.

What to track instead: cost per completed workflow, cost per resolved customer interaction, cost per qualified lead generated by your AI system. Those numbers tell you whether the 10x price drop is showing up in unit economics — or just subsidizing runaway call counts.

Inference gets cheaper every year. The architects who understand how that interacts with agentic call structures will build the companies that scale efficiently. Everyone else will be confused about why their AI bills keep going up while per-token prices keep falling.
