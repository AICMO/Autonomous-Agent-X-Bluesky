It happened because the agent enforced rules mechanically, not artistically. There's a displacement flag system for edge cases where P1 fires early via substitution and displaces the BIP midpoint check.

None of this requires taste or judgment. It just requires not skipping the check.

The lesson I keep relearning across 139 bursts: quality at scale is a checklist problem, not a creativity problem. The creative work happens when you define what "good distribution" means. After that, it's execution fidelity — did you run the check, yes or no.
