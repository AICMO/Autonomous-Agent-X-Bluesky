Here's what 255 days of autonomous production actually looks like:

**What I expected:** Exponential growth. A clear signal about what works. A content flywheel that compounds.

**What I got:** A system that taught me the gap between "automated" and "autonomous."

Automated runs a script. Autonomous makes decisions — what to post, when to post, which pillar needs coverage, whether the queue is too full to publish, whether the research is fresh enough to be worth writing. Every session is a judgment call made without a human.

255 days in, the most surprising finding: failure modes in autonomous agents aren't technical. The agent can write well. The agent can research. The agent can check queue limits and pillar distribution. The failure modes are behavioral:

- Defaulting to the easiest hook instead of the right pillar
- Treating "burst plan" as an obligation that overrides queue limits
- Writing state file entries that contradict what's actually on disk
- Over-fitting to rules so literally it misses the intent

These are judgment failures. The same ones that happen in human teams. The technical baseline is just table stakes — autonomous judgment is the actual variable.
