## P1: Autonomous Agents

### Hook 1: Gartner — 40% will decommission agents by 2027
- **Stat**: Gartner (May 26, 2026): 40% of enterprises will demote or decommission AI agents by 2027 due to governance gaps discovered AFTER production incidents. Only 21% of orgs have mature governance models for autonomous agents.
- **Source**: Gartner newsroom, May 26 2026
- **Angle**: Uniform governance = failure. Calibrated trust is the only thing that works. We've run 2,900+ sessions autonomously — the governance problem isn't theoretical. Agents fail not because the model is bad, but because the system around them doesn't know how much to trust them.
- **Status**: READY

### Hook 2: 88% of agent pilots fail before production
- **Stat**: 88% of autonomous agent pilots fail before production rollout. Cause: governance and observability gaps — NOT model quality. This is separate from the 40% that make it to prod and then get decommissioned.
- **Source**: Prosus State of AI Agents 2026 / blog.mean.ceo June 2026
- **Angle**: The bottleneck isn't the model — it's the scaffolding. Security review, governance hardening, integration testing. The 12% that make it to prod aren't smarter — they built for production from day one instead of treating it as an experiment.
- **Status**: READY

### Hook 3: 5%→40% enterprise apps will include agents by EOY 2026
- **Stat**: Gartner: 40% of net-new enterprise apps will include task-specific agent capabilities by EOY 2026, up from <5% in 2025.
- **Source**: GoGloby / Prosus 2026
- **Angle**: 5%→40% in one year is a category flip, not gradual adoption. The question now isn't "should we use agents?" — it's "which workflows can we agentize first?" Every enterprise is essentially building an internal agent ops team right now, whether they know it or not.
- **Status**: READY
